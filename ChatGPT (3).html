It looks like the logic for resampling and aggregating data in your script could benefit from a few refinements. Specifically, the problem might be related to how the resampleData function is handling individual updates, which could cause improper aggregation of bars for different timeframes.

Here’s what could be happening:

  1. Inconsistent Resampling on New Data: In your resampleData function, you are resampling based on a single piece of new data. This can cause incomplete or incorrect aggregation, especially when the new data does not fall exactly on the boundary of a new resampling interval.

  2. Issue with Accumulating Bars: You are passing a single data point [newData] to resampleData and updating the existing timeframe arrays. But for larger timeframes (e.g., "1M", "1W"), it likely isn't capturing the full picture, as each new data point is treated as its own bar.

Suggestions for Improvement:

1. Batch Resampling Instead of Individual Points

Instead of passing individual new data points ([newData]) to resampleData, consider accumulating the data over time and performing resampling on a batch basis (over several seconds or minutes).

2. Ensure Bars Accumulate Properly

When updating the higher timeframes (like "5", "15", "60"), instead of simply pushing the resampled data, you should make sure that multiple incoming ticks within the same timeframe contribute to the same bar (based on the time window).

Here's a revised resampleData function to handle this better:

Revised resampleData function:

javascript
function resampleData(data, timeframe) {
    const resampledData = [];
    let currentBar = null;
    const timeframeMs = getTimeframeMs(timeframe);

    for (const row of data) {
        const barTime = Math.floor(row.time / timeframeMs) * timeframeMs;

        // Start a new bar if the current one is missing or belongs to a different time bucket
        if (!currentBar || currentBar.time !== barTime) {
            if (currentBar) {
                resampledData.push(currentBar);
            }
            currentBar = {
                time: barTime,
                open: row.open,
                high: row.high,
                low: row.low,
                close: row.close,
                volume: row.volume
            };
        } else {
            // Update the current bar with new highs, lows, and close
            currentBar.high = Math.max(currentBar.high, row.high);
            currentBar.low = Math.min(currentBar.low, row.low);
            currentBar.close = row.close;
            currentBar.volume += row.volume; // Accumulate volume
        }
    }

    // Push the final bar if still open
    if (currentBar) {
        resampledData.push(currentBar);
    }

    return resampledData;
}
